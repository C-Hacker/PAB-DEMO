Welcome to Prisma Access Demo 3 "Upload Download"

In our Prisma Access Browser (PAB) demo, we’re highlighting powerful data protection capabilities that provide fine-grained control over copy-paste actions.

A key capability is ensuring that sensitive data downloaded onto devices is securely protected through encryption.

Scenario A

In this example, you’ll download a file containing sensitive information. When you begin the download, a warning message will appear; click "Proceed" to continue.


Click here to download the sample DLP file

In the top-right corner, click “Keep” to proceed with the download.

After downloading, open the "Download" folder. Right-click the file, select "Open with," click choose more and select Notepad.

You’ll notice the file opens in Notepad, but the data is encrypted and unreadable.

Next, open the same file within the Prisma Access Browser.

The file can only be read (decrypted) when opened with the Prisma Access Browser.

Upload a File

File uploads are permitted without scanning for sensitive data, as data is uploaded to a secure corporate applications.

However, to protect our organization, we enforce measures to ensure only legitimate files are uploaded, guarding against threats like malware, spyware, and ransomware.
