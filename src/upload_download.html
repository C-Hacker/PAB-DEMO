Welcome to Prisma Access Demo 3 "Upload Download"

In our Prisma Access Browser (PAB) demo, we’re highlighting powerful data protection capabilities that provide fine-grained control over copy-paste actions.



A key capability is ensuring that sensitive data downloaded onto devices is securely protected through encryption.

Example A

In this example, you’ll download a file containing sensitive information. When you begin the download, a warning message will appear; click "Proceed" to continue.


Click here to download the sample DLP file

After downloading, right-click the file, select "Open with," and choose Notepad. You’ll notice the file opens in Notepad, but the data is encrypted and unreadable.

Next, open the same file within the Prisma Access Browser. Here, the data is decrypted seamlessly, making it fully readable.

File uploads are permitted, as data is directed to secure corporate applications. However, to protect our organization, we enforce measures to ensure only legitimate files are uploaded, guarding against threats like malware, spyware, and ransomware.
