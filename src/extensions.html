Extensions

This page can be used to showcase Prisma Access Browser capabilities to control Extensions installed in the web browser.


Gramarly is the only extension allowed in this demo.

Try to add/install an extension:

  • 1. Click on settings (tree dots in the upper right corner of you web browser)
  • 2. Click Extensions > Visit Chrome Web Store
  • 3. Search for an extension; for example "Chat GPT"
  • 4. Open the extension and try to add/install the extension. Note: you are not allowed ;)
